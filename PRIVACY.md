# Status Window Privacy Policy

Last updated: June 29, 2026

Status Window is a study and focus tracking Chrome extension. This policy describes what information the extension stores, why it stores it, and how that information is used.

## Information collected

Status Window stores:

- Account information you provide through Supabase authentication, such as your email address and display name.
- Study data you create in the extension, including subjects, session logs, timer history, streak-related stats, and settings.
- Optional friend data used for the Friends feature, including friend request relationships and the study activity summaries needed to show current friend status.
- Local extension state needed for core functionality, such as active timer state, timer completion notices, and cached settings.

Status Window does not collect web browsing history, page contents, passwords, payment information, or unrelated website activity.

## How information is used

Status Window uses collected information only to provide the extension's user-facing features, including:

- signing you in,
- saving and syncing your study sessions and subjects,
- calculating your stats, streaks, HP, and level,
- remembering your settings,
- powering optional friend and side panel timer features.

Status Window does not sell personal information and does not use collected data for advertising.

## Where information is stored

- Study and account data are stored in the Supabase project configured for the extension.
- Some temporary operational data is stored locally in Chrome extension storage so timers and notifications continue working while the popup is closed.
- Authentication session material is kept in session-scoped extension storage so it is cleared when the browser session ends instead of being left in persistent page-local storage.

## Sharing

Status Window only shares data with the services needed to run the extension:

- Supabase, for authentication and application data storage.

If you use the Friends feature, limited study-related information is visible to accepted friends inside the extension, such as display name, whether you are actively studying, and today's study time summary.

## Security

Status Window communicates with its backend over HTTPS and WSS connections to the configured Supabase project.

## Your choices

You can:

- stop using the extension at any time,
- sign out,
- remove the extension,
- delete or change your own study data through the extension and your Supabase project controls.
